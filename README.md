# Rudransh Singh Tomar

Blockchain developer and security researcher.

I work as a blockchain developer and Security at Prospective LLC and I co-founded Trust Protocol, an undercollateralized lending layer that uses game theory to make cheating cost more than it can ever pay out. Most of my time goes into writing production Solidity and Rust, auditing contracts the way an attacker would think about them, and designing protocols from the architecture up. Along the way I have written more than a hundred technical design documents that cover everything from perpetual exchange settlement to ZK privacy and oracle design.

## What I have worked on

- Built **Trust Protocol**, undercollateralized lending across Solidity and Sui
- Blockchain development at **QuadB Technologies**, mostly DeFi and Internet Computer infrastructure
- Auditing and securing smart contracts with Foundry, Slither, Echidna, and Halmos at ProspEctive LLC
- Protocol research and proposal work across Sui, Algorand, Oasis, and other ecosystems
- Designed a Tranche Protocol that integrates with Accountable, Upshift & Middas protocol. 

## Selected work

- **Hush Protocol** ([repo](https://github.com/trudransh/hush-ui)). Trustless wallet recovery on the Internet Computer using DKIM signatures and VetKeys. Won a $25K grant at Encode x ICP.
- **Trust Protocol** ([repo](https://github.com/trudransh/trust-bonds)). A Web3 trust layer for undercollateralized lending. Won the professional track at SUI Hacker House Kathmandu.
- **Sentinel Protocol** ([repo](https://github.com/trudransh/Sentinal)). A programmable policy layer for AI-agent treasuries on Solana, so an autonomous agent can only do what it is explicitly allowed to.
- **Blockchain in Rust** ([repo](https://github.com/trudransh/Blockchain--Rust)). A blockchain built from scratch to really understand the internals.
